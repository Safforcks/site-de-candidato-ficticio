# 🗳️ Site de Campanha

Site de campanha política desenvolvido com **React** e **Vite**, criado para apresentar o candidato fictício ao eleitorado de forma moderna e acessível.

## 📋 Seções

- **Header** — Navegação fixa com menu responsivo
- **Hero** — Seção de destaque com chamada principal
- **Biografia** — Trajetória e perfil do candidato
- **Propostas** — Plano de governo e principais propostas
- **Agenda** — Calendário de eventos e compromissos
- **Vídeos** — Galeria de vídeos da campanha
- **Contato** — Formulário para apoiadores entrarem em contato
- **Footer** — Rodapé com informações e links

## 🚀 Tecnologias

- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Material UI (MUI)](https://mui.com/)
- [Styled Components](https://styled-components.com/)
- [React Icons](https://react-icons.github.io/react-icons/)
- CSS Modules

## 📁 Estrutura do projeto

```
src/
├── components/       # Componentes de cada seção
├── styles/           # CSS Modules por componente
├── imagens/          # Recursos de imagem
```
